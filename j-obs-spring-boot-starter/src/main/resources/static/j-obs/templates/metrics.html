Counters: {{COUNTER_COUNT}}
Counter: Valor que só aumenta (nunca diminui). Reseta quando a aplicação reinicia.

Ex: total de requisições HTTP, número de erros, mensagens processadas.
Gauges: {{GAUGE_COUNT}}
Gauge: Valor instantâneo que pode subir ou descer. Representa estado atual.

Ex: memória heap usada, threads ativas, conexões no pool, tamanho de fila.
Timers: {{TIMER_COUNT}}
Timer: Mede duração de eventos. Combina contador + histograma de tempos.

Ex: latência de requisições HTTP, tempo de execução de queries SQL.
Distributions: {{DISTRIBUTION_COUNT}}
Distribution: Histograma de distribuição de valores. Similar a Timer mas para valores não-temporais.

Ex: tamanho de payloads, número de itens em batch.
